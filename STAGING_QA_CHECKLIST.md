# Staging 動作確認チェックリスト

デプロイ後の手動確認用です。EffectResolver リファクタ後や `develop` への push 後に使ってください。

## 0. デプロイ完了確認

- [x] GitHub Actions で `build` → `deploy-staging` が成功している
- [x] API ヘルスチェックが通る

```bash
curl https://api.teijitaisha-web-staging.mottainaigames.com/health
```

- [x] Web が開ける（404・証明書エラーなし）

```bash
curl -sI https://teijitaisha-web-staging.mottainaigames.com | head -3
```

---

## 1. ルーム・ロビー（2人）

**準備:** ブラウザ2つ（または通常＋シークレット）でアクセス

- [x] ホストがルーム作成できる
- [x] ゲストがルームコードで参加できる
- [x] ロビーに両者の名前が表示される
- [x] 入室アナウンスが出る（ゲスト参加時）
- [x] ホストのみ「ゲーム開始」が押せる
- [x] ゲストは開始ボタンが押せない

### ロビー操作（ホスト）

- [x] プレイ順 D&D で並び替えできる
- [x] シャッフルで順番が変わる
- [x] 名前変更（自分・CPU）が反映される
- [x] 追い出し（kick）が動く
- [x] 「Xで招待」リンクが開ける

---

## 2. 基本プレイ（2人・手動）

- [x] ゲーム開始後、先攻の引きフェーズになる
- [x] 左隣（ルールどおりの席）からカードを引ける
- [x] ペアを出せる（ノルマなど）
- [x] ペアを出さずスキップできる
- [x] ターンが次のプレイヤーに進む
- [x] 場の捨て札・ログが更新される
- [ ] 手札枚数が他人に見えない（自分だけ見える）

---

## 3. カード効果（優先度高）

全部やらなくてよいですが、**リファクタの影響が大きい効果**は最低1回ずつ触ると安心です。


| カード   | 確認ポイント                     | OK  |
| ----- | -------------------------- | --- |
| ノルマ   | 効果なしでターン継続                 | [ ] |
| 飲み会   | 次のターン、対象がペアを出せず終了する        | [ ] |
| エナドリ  | もう1組ペアを出せる                 | [ ] |
| タバコ休憩 | 全員のタバコ休憩が場に出る              | [ ] |
| 会議    | 残業持ちの宣言表示（該当者がいれば）         | [ ] |
| 情報共有  | 全員が渡すカードを選び、左隣へ渡る          | [ ] |
| 社内恋愛  | 2人の手札が見え、スキップ or タイムアウトで終了 | [ ] |
| 新人教育  | 相手のカードを見て、取る/取らないを選べる      | [ ] |
| 取引    | 2人がカードを選び交換する              | [ ] |
| 労基    | 相手のカードを1枚公開（残業なら即終了）       | [ ] |
| パワハラ  | 自分のカードを相手に渡す               | [ ] |


### 効果 UI

- [ ] カード効果モーダルがメニューより前面に出る（z-index）
- [ ] Esc でモーダルが閉じる（仕様どおり）
- [ ] 効果中にハンバーガーメニューが使える（プレイ順の折りたたみ・スクロール）

---

## 4. 不正操作・エッジ

- [ ] 自分の番でないときに操作してもゲームが破綻しない（エラー表示 or 無視）
- [ ] 選べない対象・カードを選んでもクラッシュしない
- [ ] タイムアウト後も進行が止まらない（放置して数ターン進む）

---

## 5. 切断・再接続

- [ ] 対局中にタブを閉じて再度同じ URL / セッションで復帰できる
- [ ] 再接続後、手札・フェーズ・待ち入力が正しい
- [ ] 相手画面に切断表示があり、復帰後に戻る

---

## 6. CPU 参加

- [ ] ロビーで CPU を追加できる
- [ ] CPU 参加でゲームが開始・進行する
- [ ] CPU のターンが自動で進む（引き・ペア or スキップ）
- [ ] 社内恋愛中、CPU は自動スキップして進む

---

## 7. 退出・終了

- [ ] 退出ボタンが赤く、2段階確認がある
- [ ] 退出後、ロビー or ホームに戻れる
- [ ] 定時退社（手札0）で退社表示・順位が付く
- [ ] 労基で残業摘発時、ゲーム終了と結果が表示される

---

## 8. オブザーバー（任意）

- [ ] 観戦参加でルームに入れる
- [ ] 観戦者は操作できない
- [ ] 手札など秘匿情報が見えない
